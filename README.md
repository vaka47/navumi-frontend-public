# Navumi Frontend (Public Clone)

Public showcase clone of the frontend for `Navumi`, a sports camp and club platform.

## Stack

- Next.js
- React
- TypeScript
- component-based UI
- Google Maps integration points

## Run locally

1. Copy `.env.example` to `.env.local`
2. Fill required public environment variables
3. Install dependencies
4. Start with `npm run dev`

## Notes

- This public clone was sanitized for open-source sharing
- Real environment values and private deployment artifacts were removed
